# default


## Clone the repo:
```
git clone https://github.com/iobio/gene.iobio
```

## Copy sample env file
```
cd gene.iobio
cp .env.template .env
```


## Project setup

```
# yarn
yarn

# npm
npm install

# pnpm
pnpm install
```

### Compiles and hot-reloads for development

```
# yarn
yarn dev

# npm
npm run dev

# pnpm
pnpm dev
```

### Compiles and minifies for production

```
# yarn
yarn build

# npm
npm run build

# pnpm
pnpm build
```
